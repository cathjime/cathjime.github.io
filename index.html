Projects

WanderList

This is a React Native Expo Application frontend with a Ruby on Rails backend. Travelers can create loose itineraries by adding locations to Favorites based on city. The user can search for locations on a map of their currrent city. The business location data is pulled from the Yelp API based on a search term entered by the user. User can create favorites and save them to a list, update them, and delete them.

MyRecipes

This application was built using a Vanilla JavaScript frontend with a Ruby on Rails backend. Users can search for recipes by selecting cuisine type and specified ingredients in addition to filtering recipes based on food sensitivity. The application uses data from the Spoonacular API to compile recipes. The design is structured using Bootstrap.
